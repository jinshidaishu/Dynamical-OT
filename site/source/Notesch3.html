Chapter 3
Entropic Regularization of Optimal Transport

Note that Kantorovich OT (??) is a linear programming with convex constraints, therefore simplex algorithm or interior point methods can be naturally adopted. However, the computational cost for these methods are \(O(N^3 \log N), N = \max \{n,m\}\), hence inefficient.

This chapter introduces a family of numerical schemes to approximate solutions to Kantorovich formulation of optimal transport (??) and its many generalizations. It operates by adding an entropic regularization penalty to the original problem. This regularization has several important advantages, which make it, when taken altogether, a very useful tool: the minimization of the regularized problem can be solved using a simple alternate minimization scheme; that scheme translates into iterations that are simple matrix-vector products, making them particularly suited to execution on GPU; for some applications, these matrix-vector products do not require storing an \(n \times m\) cost matrix, but instead only require access to a kernel evaluation; in the case where a large group of measures share the same support, all of these matrix-vector products can be cast as matrix-matrix products with significant speedups; the resulting approximate distance is smooth with respect to input histogram weights and positions of the Diracs and can be differentiated using automatic differentiation.
